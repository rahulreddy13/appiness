# appiness
Making the first user who registers, as Admin. 

● The code should be well structured
● Must have proper comments 
● Should document the workflow and how to run the code 

Description :
There should be two models Users and User_Roles.
First time when user registers, the User_Roles table should contain value as admin for that user and for rest of them it should be different.

TO Run the code :

Step 1 : git clone https://github.com/rahulreddy13/appiness.git

Step 2 : cd appiness

Step 3 : npm install 

Step 4 : npm start

You can find application running on port 8080.

URL : http://localhost:8080
